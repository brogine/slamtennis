SLAM  Login

En esta pantalla el usuario podrá realizar el Login al sistema mediante su Nombre de Usuario y Contraseña.

Cuando se ingrese al sistema por primera vez este le pedirá que cree un usuario para poder validar el login.

A continuación de la pantalla mostrada arriba, el sistema le pedirá que ingrese los datos del empleado que vaya a utilizar la aplicación. (Véase SLAM-Gestión Empleados >> Nuevo Empleado).

SLAM  Pantalla Principal

En esta pantalla el usuario podrá acceder a los diferentes sub menús que componen el sistema. Aquí se concentran todos los accesos directos a las diferentes secciones descriptas y explicadas en el siguiente manual. Esta pantalla corresponde a la versión Desktop de la aplicación.

 

 

SLAM  Archivo

Desde esta pantalla el usuario podrá tanto finalizar su sesión actual, utilizando el botón Cerrar Sesión, como salir del sistema, utilizando el botón Salir.

 

SLAM  Gestión

Desde esta pantalla el usuario podrá acceder a los diferentes sub menús comprendidos dentro de Gestión, a continuación se detalla el funcionamiento de cada uno.

Menú Jugadores >> Administrar

Desde esta pantalla el usuario podrá visualizar el listado de Jugadores pertenecientes a un determinado Club, utilizando el filtro correspondiente. Además cuenta con las funcionalidades para crear una Nueva Afiliación, Modificar cualquier Jugador que seleccionemos del listado o dar de baja un Jugador determinado.

Con el botón Nueva Afiliación accedemos a esta pantalla en la cual se podrá realizar una nueva afiliación de un jugador a un determinado club.

Con el botón Nuevo Jugador de la pantalla anterior accedemos a esta pantalla en la cual se podrá realizar el alta de un nuevo Jugador. Si se desea modificar los datos de algún Jugador simplemente se debe utilizar el botón de Modificar Jugador mostrado más arriba lo cual llevará al usuario a una pantalla idéntica a la mostrada abajo.

 

Menú Jugadores >> Estadísticas

Desde esta pantalla el usuario podrá visualizar el listado de Jugadores con sus respectivas estadísticas filtrando los datos por Club y Categoría. Además cuenta con las funcionalidades para crear una Nueva Estadística y Modificar cualquier Estadística que seleccionemos del listado.

Con el botón Nueva Estadística accedemos a esta pantalla en la cual se podrá realizar una nueva estadística de un jugador.

 

Menú Árbitros

Desde esta pantalla el usuario podrá visualizar el listado de Árbitros pertenecientes a la Federación. Además cuenta con las funcionalidades para crear un Nuevo Arbitro o Modificar uno existente.

Con el botón Nuevo Árbitro de la pantalla anterior accedemos a esta pantalla en la cual se podrá realizar el alta de un nuevo Árbitro. Si se desea modificar los datos de algún Árbitro simplemente se debe utilizar el botón de Modificar Árbitro mostrado más arriba lo cual llevará al usuario a una pantalla idéntica a la mostrada abajo.

 

Menú Empleados

Desde esta pantalla el usuario podrá visualizar el listado de Empleados pertenecientes a la Federación. Además cuenta con las funcionalidades para crear un Nuevo Empleado o Modificar uno existente.

Con el botón Nuevo Empleado de la pantalla anterior accedemos a esta pantalla en la cual se podrá realizar el alta de un nuevo Empleado. Si se desea modificar los datos de algún Empleado simplemente se debe utilizar el botón de Modificar Empleado mostrado más arriba lo cual llevará al usuario a una pantalla idéntica a la mostrada abajo.

 

Menú Categorías

Desde esta pantalla el usuario podrá visualizar el listado de Categorías pertenecientes a la Federación. Además cuenta con las funcionalidades para crear una Nueva Categoría o Modificar una existente.

Con el botón Agregar Categoría de la pantalla anterior accedemos a esta pantalla en la cual se podrá realizar el alta de una nueva Categoría.

Con el botón Modificar Categoría de la pantalla anterior accedemos a esta pantalla en la cual se podrá realizar la modificación Categoría existente.

Menú Ubicaciones

Desde cualquier formulario en el que el usuario necesite ingresar una ubicación podrá utilizar el botón Nueva Localidad.

 En esta pantalla el usuario podrá escribir el nombre del País, Provincia y Localidad y a continuación utilizar el botón Agregar País, Agregar Provincia y Agregar Localidad respectivamente.

SLAM  Clubes

Desde esta pantalla el usuario podrá acceder a los diferentes sub menús comprendidos dentro de Clubes, a continuación se detalla el funcionamiento de cada uno.

Menú Clubes >> Administrar

Desde esta pantalla el usuario podrá visualizar el listado de Clubes pertenecientes a la federación. Además cuenta con las funcionalidades para crear un Nuevo Club o Modificar uno existente.

Con el botón Nuevo Club accedemos a esta pantalla en la cual se podrá realizar el alta de un nuevo club.

Con el botón Modificar Club accedemos a esta pantalla en la cual se podrá realizar el alta de un nuevo club.

 

Menú Clubes >> Sedes

Desde esta pantalla el usuario podrá visualizar el listado de Sedes pertenecientes a un determinado Club, utilizando el filtro correspondiente. Además cuenta con las funcionalidades para crear una Nueva Sede,  Modificar una Sede existente, crear una nueva Cancha para una determinada Sede o Modificar una existente.

Con el botón Nueva Sede de la pantalla anterior accedemos a esta pantalla en la cual se podrá realizar el alta de una nueva Sede.

Con el botón Modificar Sede de la pantalla anterior accedemos a esta pantalla en la cual se podrá realizar la modificación de una Sede existente.

Con el botón Nueva Cancha de la pantalla anterior accedemos a esta pantalla en la cual se podrá realizar el alta de una nueva Cancha para una Sede en particular.

SLAM  Eventos

Desde esta pantalla el usuario podrá acceder a los diferentes sub menús comprendidos dentro de Eventos, a continuación se detalla el funcionamiento de cada uno.

Menú Torneos >> Administrar

Desde esta pantalla el usuario podrá visualizar el listado de Torneos. Además cuenta con las funcionalidades para Agregar Torneo o Modificar un Torneo existente.

Con el botón Agregar Torneo de la pantalla anterior accedemos a esta pantalla en la cual se podrá realizar el alta de un nuevo Torneo.

Con el botón Modificar Torneo de la pantalla anterior accedemos a esta pantalla en la cual se podrá realizar la modificación de un Torneo existente.

 

Menú Torneos >> Partidos

Desde esta pantalla el usuario podrá visualizar el listado de los Partidos correspondiente a un Torneo en particular. Además cuenta con las funcionalidades para Agregar un Partido o Modificar un Partido existente.

Menú Torneos >> Inscripciones

Desde esta pantalla el usuario podrá visualizar el listado de las Inscripciones correspondientes a un Torneo en particular. Además cuenta con las funcionalidades para crear una Nueva Inscripción, Modificar una Inscripción existente o Eliminarla.

Con el botón Nueva de la pantalla anterior accedemos a esta pantalla en la cual se podrá realizar el alta de una nueva Inscripción para un Torneo previamente seleccionado.

 

*Notificaciones

Desde la pantalla principal, tal cual se muestra en la captura anterior, el usuario podrá visualizar y acceder a las notificaciones correspondientes al estado de los diferentes Torneos activos y de los cuales se necesite completar el resultado de los partidos correspondientes a este.

 

Con el botón Completar de la pantalla anterior accedemos a esta pantalla en la cual se podrán completar las resultados de los diferentes partidos correspondientes al Torneo en cuestión.

*Para más información ver sección Menú Torneos >> Partidos

 

SLAM  Reportes

Desde esta pantalla el usuario podrá acceder a los diferentes sub menús comprendidos dentro de Reportes, a continuación se detalla el funcionamiento de cada uno.

Menú Reportes

Con el botón Ranking de la pantalla anterior accedemos a esta pantalla en la cual se podrá visualizar el reporte de Ranking por Categoría, la cual es seleccionada previamente por el usuario.

A continuación se muestra un ejemplo del reporte generado anteriormente utilizando los parámetros descriptos:

 

Con el botón Carnet de la pantalla anterior accedemos a esta pantalla en la cual se podrá seleccionar el tipo de carnet que queramos generar, es decir los diferentes tipos de personas que componen el sistema, Arbitro, Empleado, Jugador, seguido del número de DNI correspondiente. Luego de completados estos datos se generará el reporte mostrando el carnet del usuario.

Desde el menú de Partidos se podrá elegir cualquier torneo activo de la lista y luego utilizar el botón de Generar Llave lo cual nos mostrará un detalle de los partidos que conforman dicho Torneo en forma de llaves.

SLAM  Configuración

Desde esta pantalla el usuario podrá acceder a la configuración de la aplicación, mas precisamente a los parámetros de conexión a la base de datos que se va a utilizar, en la actual versión se permiten conexiones tanto a SqlServer como a MySql. Desde aquí el usuario podrá elegir cual utilizar así como el nombre del servidor donde se alojara la base de datos en cuestión. También se cuenta con un botón de prueba para corroborar el buen funcionamiento de la conexión previamente seleccionada.

 

SLAM  Ayuda

Desde esta pantalla el usuario podrá acceder a la ayuda de la aplicación y a los datos referidos a la versión actual del sistema. En cuanto a el botón Manual HTML, este lo guiará hacia la versión mas actual de este manual.  

SLAM  Web Access

A continuación se muestran las funcionalidades web del sistema, es decir aquellas a la que los usuarios principalmente Jugadores de la Federación, podrán ingresar a través de su explorador web mediante un Login. Desde aquí el usuario tendrá diferentes herramientas tales como ver sus datos estadísticos, historiales, llaves de torneos o incluso registrarse para algún torneo que se encuentre disponible.

Menú Login

En esta pantalla el usuario podrá realizar el Login al sistema mediante su Nombre de Usuario y Contraseña.

 

Menú Principal

En esta pantalla el usuario podrá acceder a los diferentes sub menús que componen el sistema web.

 

Datos Estadísticos

En esta pantalla el usuario puede ver los reportes correspondientes a los rankings de las diferentes categorías que posee la Asociación. Simplemente se selecciona la Categoría deseada en el primer combo y luego se elige el Tipo de Categoría, es decir Single o Dobles.

Desde el link de la esquina inferior derecha de la pantalla anterior el usuario puede rápidamente ver en pantalla sus datos estadísticos, es decir Partidos Jugados, Ganados, Perdidos , Torneos Jugados, Completados y sus Puntos en la categoría actual en la que se encuentra, tanto para Single como Dobles. La información se muestra como en la imagen de abajo.

 

Mis Datos

 

En esta pantalla el usuario puede ver los datos que el sistema posee de si mismo, en caso de querer corroborarlos y pedir alguna modificación en caso de ser necesaria, para lo cual algún Empleado de la Federación con acceso a la versión Desktop del Sistema podrá hacerlo.

 

Llaves Torneos

En esta pantalla el usuario puede ver los reportes correspondientes a las Llaves de Torneos activos, es decir el detalle con los diferentes enfrentamientos y resultados de los partidos. Solo deberá seleccionar el Torneo que desea ver y a continuación se mostrará un reporte dando la información de los partidos jugados hasta el momento.

 


Torneos

En esta pantalla el usuario puede ver los Torneos que hay disponibles en el sistema, en este listado se mostrarán los datos más importantes de cada torneo, fechas, inscripciones, estado, etc. También el usuario podrá ver si se encuentra inscripto o no a dicho torneo, en caso de no estarlo este puede inscribirse directamente desde la web siempre y cuando este se encuentre abierto para la inscripción.

Desde el link de la columna Inscripto? el usuario puede acceder a la pantalla mostrada abajo donde puede ver los datos que va a utilizar para su inscripción y posteriormente validar dicha inscripción con el link Inscribirse.

En caso de estar inscripto a un determinado Torneo, el usuario puede dar de baja dicha inscripción si por algún motivo no participará del Torneo. Solo debe acceder al link Borrar? y verá el siguiente mensaje para completar con la baja.

Mensajes al Administrador

En esta pantalla el usuario puede enviar cualquier mensaje que desee al administrador del sistema, en este caso algún empleado designado por la Federación, esta herramienta podrá ser utilizada en caso de requerir algún tipo de información, consultas sobre torneos, actualizar datos de algún jugador, etc.

 

Cerrar Sesión

En esta pantalla el usuario podrá cerrar su sesión actual y ser redirigido a la pantalla de inicio.
